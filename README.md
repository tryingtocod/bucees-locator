# Buc-ee's Locator

Simple static site that plots Buc-ee's locations on a Leaflet map.

## What this repo contains
- `public/` — production-ready site (open `public/index.html` in a browser or serve it)
- `src/` — source files (copies of JS/CSS/data)
- `ATTRIBUTION.md` — trademark/logo usage note
- `LICENSE` — MIT license

## Running locally
You can serve the `public/` folder with any static server. Example using `http-server`:

```bash
npx http-server public -p 8080
# then open http://localhost:8080
```
